## Dataset Information

The dataset contains student academic records and performance-related attributes. Each record includes study habits, attendance information, assignment completion rates, previous academic results, and the final exam score used as the target variable.

---

## Dashboard Features

The dashboard provides:

* Total number of records
* Average marks
* Average attendance
* Attendance distribution visualization
* Marks distribution visualization
* Feature correlation heatmap
* Performance category analysis



## Future Improvements

Some possible future enhancements include:

* Training additional machine learning models such as Random Forest and XGBoost
* Real-time prediction through a Flask/FastAPI backend
* User authentication and student profile management
* Deployment on cloud platforms
* Improved visual analytics and reporting
* Integration with educational management systems
